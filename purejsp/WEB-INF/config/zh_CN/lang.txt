#project=PureJSP

index_desc=<h3> \
PureJSP是一个轻量级的Java网页应用程序。全部代码使用JSP格式文件开发。只需要复制purejsp文件夹到服务器webapps目录下运行即可看到效果，所有东西都是那么简单。除了mysql.jar没有其它jar，更不需要maven。<br> \
<br> \
(开发环境 : JDK 1.6 + Tomcat 6 + MySql 5.1  ) <br> \
MIT License<br><br> \
</h3> \
<pre> <br> \
Setup <br> \
1, 复制purejsp文件夹到服务器的webapps目录 <br> \
2, MySQL服务器上运行数据库初始化脚本“{purejsp}/web-inf/config/zh_CN/db_init.txt” <br> \
3, 编辑数据库配置文件“{purejsp}/web-inf/config/db_config.txt” <br> \
4, 启动web服务器，打开浏览器定位“localhost:8080/purejsp/index.jsp” <br> \
5, 建议使用Chrome或Firefox浏览器进行查看。 <br> \
</pre> <br> \
