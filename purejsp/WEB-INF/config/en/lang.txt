index_desc=<h3> \
PureJSP is a java web application and all files are written in JSP format.<br> \
Just copy all the files to your webapps directory and run, then everything will be easy !<br> \
<br> \
(Test on : JDK 1.6 + Tomcat 6 + MySql 5.1  ) <br> \
MIT License<br><br> \
</h3> \
<pre> <br> \
Setup <br> \
1, Copy the purejsp folder to the webapps directory <br> \
2, Run the database script "source {purejsp}/WEB-INF/config/en/db_init.txt" on your mysql server <br> \
3, Edit the database configuration file "{purejsp}/WEB-INF/config/db_config.txt" <br> \
4, Start Web Server , open "localhost:8080/purejsp/index.jsp" <br> \
5, Chrome or Firefox browser is recommended for viewing  <br> \
</pre> <br> \
